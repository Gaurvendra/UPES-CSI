Workshops

Domain specific and some career-oriented workshops organised by UPES-CSI that trains you to learn emerging technologies and upskill yourself.

Python Workshop

UPES-CSI conducted a workshop on "Python Programming" in collaboration with 'Coding Blocks'. Python is one of the most trending programming languages that uses Object-oriented approach to solve real world problems. The workshop covered some basic concepts of OOPS, real world usecases and what makes Python different from other languages. Some basic ideas regarding project development using Python was also provided to the attendees.











Design Workshop

“Design can be art. Design can be aesthetics. Design is so simple, that's why it is so complicated." UPES-CSI organized a "Design Workshop", where participants learned some basic use cases of "Adobe Photoshop" and how this tool can be used efficiently. Along with that, our mentors gave some real-world insights to the participants and helped them in creating some templates using Photoshop.











Linux Workshop

Almost every tech organization uses Linux System because of its highly popular file system and security that it provides. Keeping that in mind, UPES-CSI organized a workshop on "Linux and its commands" to help the students to get aware of this operating system and its working. Our specialized mentors focused on the basic commands and how the command-line interface(CLI) is superior to graphical user interface(GUI) and how this operating system can serve as a perfect platform for making secure projects.











Web Devlopment Workshop

UPES-CSI conducted a workshop on "The basic concepts of Web Development." The main focus was on HTML5, CSS3 and Javascript and how these languages help in making responsive websites. The power of Bootstrap in web designing was also given to the attendees. The target audience was 1st and 2nd-year students and our mentors gave some real-life ideas which can be implemented by web technologies.
